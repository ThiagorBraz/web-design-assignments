# Web Design Assignments

HTML, CSS and JavaScript assignments developed for the Web and UI 
Design module at Atlantic Technological University (ATU), Sligo, 
Ireland.

## Assignments

### Assignment 1 — Games Page (HTML and CSS)
A static HTML page about video games featuring:
- Navigation links to Nintendo and Xbox websites
- Embedded video player
- Images and styled sections
- Radio buttons for game selection
- Contact link via email
- Built with HTML and CSS only

### Assignment 2 — Interactive Page (jQuery and JavaScript)
An interactive web page featuring:
- jQuery hover effects with colour transitions on elements
- Hover effects on images with fadeIn and fadeOut text
- Form with name input and dessert selector using JavaScript
- Drag and drop functionality for images into drop zones
- Built with HTML, CSS, JavaScript and jQuery

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery 3.6.0

## Academic Context

Developed at Atlantic Technological University (ATU), Sligo — 
Higher Diploma in Computer Science, 1st Semester, 
Web and UI Design Module.
